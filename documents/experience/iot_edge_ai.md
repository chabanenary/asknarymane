# IoT & Edge AI Experience

## Embedded Systems as IoT Foundation
Narymane's entire career has been centered on what is now called IoT and edge infrastructure — building the embedded firmware and system software that runs on network devices deployed at the edge of telecom networks.

## Hardware Experience
- ARM-based SoCs: Freescale/NXP QorIQ T1040, P1021, MIPS processors
- x86 embedded platforms
- Devices with Ethernet, Wi-Fi, 4G/LTE, optical SFP interfaces
- NVIDIA Jetson Nano for GPU-accelerated edge inference
- Comfortable reading hardware schematics and datasheets to write drivers
- Managed board bring-up end to end: from schematics to system validation

## Operating Systems
- Linux: Debian, Ubuntu, Yocto/OpenEmbedded (kernel configuration, patching, device tree, custom images)
- VxWorks RTOS: real-time multitasking, interrupt handling, IPC
- U-Boot bootloader customization, firmware update mechanisms, secure boot concepts
- systemd service management, custom init sequences for constrained boot environments

## Networking & Protocols
- Deep experience with TCP/IP, xDSL (ADSL/VDSL), Ethernet switching, Wi-Fi, 4G/LTE
- DHCP, DNS, HTTP/REST, RTSP, SIP, MQTT
- Multi-service devices handling data, voice, and video simultaneously

## Edge AI (since 2024)
- YOLO model training using Ultralytics framework
- Model optimization pipeline: PyTorch → ONNX → TensorRT (Jetson), TFLite (Android)
- Real-time video pipelines: GStreamer, RTSP, OpenCV on Ubuntu Linux
- Deployment under strict GPU, thermal, and power constraints on Jetson Nano
- On-device inference on Android with TFLite
