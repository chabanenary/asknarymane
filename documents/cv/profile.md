Narymane is an embedded Linux software engineer with over 12 years of R&D experience building production-grade firmware and system software for telecom-grade CPE devices. She has deep expertise in Linux system programming (Debian, Yocto/OpenEmbedded, kernel-level development), bootloaders (U-Boot), low-level hardware drivers, and RTOS (VxWorks).

She is a strong C/C++ developer with hands-on experience across the full embedded software lifecycle — from board bring-up and BSP development to CI/CD integration, automated testing, and offshore team coordination. She is a passionate open source contributor with published projects on GitHub.

More recently, she has been developing skills in ML deployment and MLOps: serving models via FastAPI REST APIs, containerization with Docker, and Kubernetes/OpenShift deployment.

She is currently building hands-on experience with LLM serving (Ollama, Mistral, Qwen), RAG pipelines (LangChain, ChromaDB, embeddings), and AI agents with MCP (Model Context Protocol) tool calling — extending her AI skillset from edge inference to complete AI-powered applications.
